Automated Waste Management & Recycling Optimizer

How to run:
1. Open Dev C++ or Visual Studio.
2. Create a new C++ Console Project.
3. Copy the code from main.cpp into your project.
4. Compile and Run.

Files automatically created by the program:
- bins.txt              : stores smart bin records
- trucks.txt            : stores garbage truck records
- collections.txt       : stores collection history
- waste_report.txt      : stores waste classification reports
- system_report.txt     : stores final system reports

OOP Concepts included:
- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Constructor and Destructor
- Function Overloading
- File Handling

Recommended compiler setting:
- Use C++11 or later.
